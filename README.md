# Python TOTAL 🐍

Este repositorio recopila las diferentes actividades realizadas en el curso online de "Python TOTAL - Programador Avanzado en 16 días" de Udemy, dictado por Federico Garay.

## 📁 Estructura del repositorio

```
📁 python-total
│
├─ 📁 D1
│   └─ proyecto-dia-1.ipynb
│   └─ apuntes-dia-1.md
│
├─ 📁 D2
│   └─ proyecto-dia-2.ipynb
│   └─ apuntes-dia-2.md
│
├── 📁 D3
│   └── ...
└─ README.md
```

Cada carpeta corresponde a un día de contenido del curso y contiene el archivo del proyecto ``.ipynb`` junto con un archivo de apuntes ``.md``.
